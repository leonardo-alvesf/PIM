Banco de Dados

Qual característica NÃO é típica de bancos de dados NoSQL?

Escolha uma alternativa

Relacionamentos complexos com JOINs são característicos de bancos SQL relacionais. Bancos NoSQL evitam JOINs em favor de desnormalização e estruturas de dados embutidas.

Banco de Dados

Qual processo organiza os dados de um banco relacional para eliminar redundâncias e dependências indesejadas?

Escolha uma alternativa

Normalização é o processo de organizar os dados dividindo tabelas grandes em menores e definindo relacionamentos entre elas, reduzindo redundâncias e garantindo integridade dos dados.
